Reality check

Powerful by design. Supervision required by design.

This project gives ChatGPT a local control surface. That is the point — and the risk. It is not a cloud bot, not a hosted service and not a security sandbox. Give it only the access you are willing to supervise.

Local-first: runs against your own machine and setup.

Scoped: use workspace boundaries and permission modes intentionally.

Visible: prefer inspect → backup → act → verify.

Honest: local power needs local responsibility.
